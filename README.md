# AttachToScreen
屏贴    
   ——把文字或图片贴在屏幕上
   
支持不限数量的文字图片同时贴在屏幕上。    
    
* 文字支持调节字号、段落宽度、透明度、置顶。无背景。
* 图片支持调节缩放、水平翻转、动图速度、透明度、置顶。无背景，支持透明png和gif。

提供项目保存，可以记录贴的情况并存到文件中。
